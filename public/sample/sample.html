変数の基礎

ローカル変数
小文字またはアンダースコアで始まる変数。メソッドやブロック内でのみ有効。
name = "Ruby" age = 30 user_score = 100
インスタンス変数
@で始まる変数。特定のオブジェクトに紐づいて状態を保持。
class Person def initialize(name) @name = name end end
クラス変数
@@で始まる変数。クラスとそのインスタンス間で共有。
class Car @@count = 0 def initialize @@count += 1 end end
定数
大文字で始まる変数。値を変更しないデータに使用。
PI = 3.14159 MAX_SIZE = 100 API_VERSION = "v2.1"

文字列メソッド

length / size
文字列の文字数を取得
"Hello".length # => 5 "Ruby".size # => 4
upcase / downcase
大文字・小文字変換
"hello".upcase # => "HELLO" "WORLD".downcase # => "world"
strip
前後の空白を除去
" hello ".strip # => "hello"
gsub
文字列の置換（全て）
"hello".gsub("l", "x") # => "hexxo"
split
文字列を分割して配列に
"a,b,c".split(",") # => ["a", "b", "c"]
include?
指定した文字列が含まれているか
"hello".include?("ll") # => true

配列メソッド

push / <<
配列の末尾に要素を追加
[1, 2].push(3) # => [1, 2, 3] [1, 2] << 3 # => [1, 2, 3]
each
各要素に対して処理を実行
[1, 2, 3].each { |n| puts n }
map
各要素を変換して新配列作成
[1, 2, 3].map { |n| n * 2 } # => [2, 4, 6]
select
条件に合う要素を抽出
[1, 2, 3, 4].select(&:even?) # => [2, 4]

ハッシュメソッド

keys
全てのキーを配列で取得
{a: 1, b: 2}.keys # => [:a, :b]
values
全ての値を配列で取得
{a: 1, b: 2}.values # => [1, 2]
each
キーと値のペアに対して処理
{a: 1, b: 2}.each do |k, v| puts "#{k}: #{v}" end
merge
2つのハッシュを結合
{a: 1}.merge({b: 2}) # => {a: 1, b: 2}

データ型の基礎

String（文字列）
文字の連続。シングル・ダブルクォートで作成。
name = "Ruby" message = 'Hello'
Integer（整数）
小数点を含まない数値。
age = 25 count = 100
Array（配列）
複数の要素を順序付けて格納。
fruits = ["apple", "banana"] numbers = [1, 2, 3]
Hash（ハッシュ）
キーと値のペアを格納。
person = {name: "Alice", age: 30}

数値メソッド

even? / odd?
偶数・奇数の判定
4.even? # => true 5.odd? # => true
abs
絶対値を取得
(-10).abs # => 10
round
四捨五入
3.7.round # => 4 3.14159.round(2) # => 3.14

その他の重要メソッド

puts
画面に出力（改行付き）
puts "Hello, World!"
gets.chomp
ユーザー入力を取得
name = gets.chomp puts "Hello, #{name}!"
nil?
nilかどうかを判定
value = nil value.nil? # => true
class
オブジェクトの型を確認
"hello".class # => String 123.class # => Integer
